# 行为准则

## 我们的承诺

为了营造一个开放、包容的社区，我们承诺：
- 对所有用户保持友好和欢迎的态度
- 不容忍任何形式的骚扰行为
- 尊重不同的观点和经验
- 接受建设性的批评 gracefully
- 关注对社区最有利的事情

## 我们的标准

有助于创造积极环境的行为包括：
- 使用友好、包容的语言
- 尊重不同的观点和经历
- 优雅接受建设性的批评
- 关注社区共同利益
- 对其他社区成员表现出同理心

不可接受的行为包括：
- 使用性暗示的语言或图像
- 骚扰行为
- 侮辱/贬低评论
- 个人或政治攻击
- 公开或私下骚扰
- 未经许可发布他人私人信息
- 其他违反道德标准的行为

## 我们的责任

社区维护者有责任：
- 明确可接受行为的标准
- 采取适当和公平的行动回应违规行为
- 移除、编辑或拒绝不符合本准则的评论、提交、代码、维基编辑、问题和其他贡献
- 暂时或永久禁止任何违反本准则的贡献者

## 适用范围

本行为准则适用于：
- 所有社区空间
- 代表社区时的公共场合

## 执法

滥用、骚扰或其他不可接受行为的举报，请联系项目维护者。所有投诉将及时、公平地审查和调查。

## 处罚指南

社区维护者将决定对违反本准则的行为采取以下行动：
1. 口头警告
2. 书面警告
3. 临时禁止
4. 永久禁止
